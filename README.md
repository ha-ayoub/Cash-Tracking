#💰 Budget Tracker App

**📌Overview**  
Budget Tracker est une application web full stack permettant de suivre ses finances en temps réel.

Elle permet aux utilisateurs de :

    gérer leurs revenus et dépenses
    visualiser leur solde
    analyser leurs habitudes financières

  ✨ Features
      ➕ Ajouter / supprimer des transactions
      💰 Calcul automatique du solde
      📊 Suivi des dépenses
      📅 Historique des transactions
      🔍 Recherche & filtres
      🔐 Authentification (JWT)
      📈 Graphiques (optionnel)
